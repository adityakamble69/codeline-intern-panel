# 🚀 COD Internship Management System

A **web-based internship management platform** designed to automate task assignment, tracking, and performance evaluation of interns.

---

## 📌 Project Overview

The **COD Internship Management System** helps organizations efficiently manage interns by providing a centralized digital platform.

### 🎯 Objective
- Eliminate manual tracking
- Improve productivity
- Automate internship workflows
- Provide real-time insights and analytics

---

## 🧩 System Architecture

- **Frontend:** HTML, CSS, JavaScript  
- **API Layer:** JavaScript (app.js)  
- **Backend:** Google Apps Script  
- **Database:** Google Sheets  

---

## 👑 Modules

### 🔹 Admin Panel
- Manage interns
- Create batches (max 20 interns)
- Assign tasks (batch-wise or individual)
- Review submissions
- Approve/Reject tasks with grades & remarks
- Track incomplete interns
- Analytics dashboard
- Handle edit permission requests
- Chat with interns

---

### 👨‍💻 Intern Panel
- View daily assigned tasks
- Submit tasks (GitHub / deployed links)
- View task history with grades
- Request edit permissions
- Chat with admin
- Manage profile (bio, avatar)
- Track performance ranking
- View batch interns
- Activity heatmap tracking

---

## 🔐 Authentication System
- Admin login
- Intern login
- First-time password setup
- OTP-based password reset
- Session-based authentication
